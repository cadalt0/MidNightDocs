# Quick Guide

Quick guides for developers moving to Midnight.

## EVM Developers

If you are coming from Ethereum/EVM development, start here:

- [EVM Developer Quick Guide](https://github.com/cadalt0/MidNightDocs/blob/main/quickguide/evm/docs/evm.md)

### What you will learn

- How Midnight differs from transparent EVM execution
- Zero-knowledge proof based transaction model
- Public ledger state vs local private state
- Wallet model and DUST workflow
- Compile and deployment mental model shifts
